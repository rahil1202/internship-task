## üß© The Anatomy of Table Layout

A table is a grid system made up of **rows (`<tr>`)** and **cells (`<td>` or `<th>`)**.
But between those cells, there are two invisible things that affect how your table *feels*:

* **Cell padding** ‚Äî space *inside* a cell (between the text and its border)
* **Cell spacing** ‚Äî space *between* cells (the gap separating them)

Let‚Äôs see how they work.

---

## üìè Cell Padding (`cellpadding`)

Padding controls **the breathing room inside each cell**.
It makes content easier to read by not squishing text against the border.

### Example:

```html
<table border="1" cellpadding="10">
  <tr>
    <th>Language</th>
    <th>Creator</th>
    <th>Year</th>
  </tr>
  <tr>
    <td>Python</td>
    <td>Guido van Rossum</td>
    <td>1991</td>
  </tr>
  <tr>
    <td>JavaScript</td>
    <td>Brendan Eich</td>
    <td>1995</td>
  </tr>
</table>
```

Each cell now has 10 pixels of space inside ‚Äî comfortable and readable.

üëâ **Modern way:** use CSS instead of `cellpadding`:

```css
td, th {
  padding: 10px;
}
```

---

## üìê Cell Spacing (`cellspacing`)

Spacing adds **gaps between cells**.
By default, there‚Äôs a small spacing ‚Äî but you can control it.

### Example:

```html
<table border="1" cellspacing="10">
  <tr>
    <th>Item</th>
    <th>Price</th>
    <th>Stock</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>$1200</td>
    <td>15</td>
  </tr>
  <tr>
    <td>Monitor</td>
    <td>$250</td>
    <td>40</td>
  </tr>
</table>
```

Cells are now separated by 10px gaps ‚Äî visible and airy.

üëâ **Modern CSS equivalent:**

```css
table {
  border-spacing: 10px;
}
```

---

## üß± `border-collapse` ‚Äî The Big Switch

When you style tables with CSS, you can either:

* Keep borders separate (`border-collapse: separate`)
* Merge them together (`border-collapse: collapse`)

### Example:

```html
<style>
table {
  width: 70%;
  border: 2px solid #333;
  border-collapse: collapse; /* or 'separate' */
}
td, th {
  border: 1px solid #333;
  padding: 10px;
}
</style>

<table>
  <tr>
    <th>Country</th>
    <th>Continent</th>
  </tr>
  <tr>
    <td>India</td>
    <td>Asia</td>
  </tr>
  <tr>
    <td>Brazil</td>
    <td>South America</td>
  </tr>
</table>
```

* `collapse`: Borders merge into a single neat line (most common).
* `separate`: Each cell‚Äôs border stands alone with spacing in between.

---

## üîÑ Aligning Content

You can control text alignment both **horizontally** and **vertically** inside cells.

### Example:

```html
<style>
table {
  width: 60%;
  border-collapse: collapse;
}
th, td {
  border: 1px solid #555;
  padding: 10px;
  text-align: left; /* horizontal: left, center, right */
  vertical-align: middle; /* vertical: top, middle, bottom */
}
</style>

<table>
  <tr>
    <th>Product</th>
    <th>Details</th>
  </tr>
  <tr>
    <td>Phone</td>
    <td>Latest Android model<br>64MP Camera<br>5G Support</td>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>16GB RAM<br>SSD 512GB</td>
  </tr>
</table>
```

## üßÆ Merging Cells with `rowspan` and `colspan`

Sometimes, you want a cell to stretch across multiple rows or columns.

### Example 1: `colspan`

```html
<table border="1" style="border-collapse: collapse;">
  <tr>
    <th colspan="3">Monthly Report</th>
  </tr>
  <tr>
    <th>Month</th>
    <th>Sales</th>
    <th>Profit</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$5000</td>
    <td>$1500</td>
  </tr>
</table>
```

The first header cell spans across 3 columns.

---

### Example 2: `rowspan`

```html
<table border="1" style="border-collapse: collapse;">
  <tr>
    <th rowspan="2">Department</th>
    <th colspan="2">Employees</th>
  </tr>
  <tr>
    <th>Male</th>
    <th>Female</th>
  </tr>
  <tr>
    <td>HR</td>
    <td>3</td>
    <td>5</td>
  </tr>
  <tr>
    <td>Tech</td>
    <td>8</td>
    <td>2</td>
  </tr>
</table>
```

The ‚ÄúDepartment‚Äù header spans two rows.

## üíÖ Full Styling Example ‚Äî Modern Table

Let‚Äôs make it beautiful and complete:

```html
<style>

  font-family: 'Segoe UI', sans-serif;
  background-color: #f8f9fa;
  padding: 20px;
}

  width: 80%;
  margin: auto;
  border-collapse: collapse;
  border: 2px solid #333;
  border-radius: 8px;
  overflow: hidden;
}

  border: 1px solid #aaa;
  padding: 12px 15px;
  text-align: center;
}

th {
  background-color: #4a4e69;
  color: white;
}

tr:nth-child(even) {
  background-color: #f2f2f2;
}

caption {
  caption-side: top;
  font-size: 1.3em;
  font-weight: bold;
  margin-bottom: 10px;
}
</style>

<table>
  <caption>Student Grades Table</caption>
  <tr>
    <th>Student</th>
    <th>Subject</th>
    <th>Grade</th>
  </tr>
  <tr>
    <td rowspan="2">Rahil</td>
    <td>Math</td>
    <td>A</td>
  </tr>
  <tr>
    <td>Science</td>
    <td>B+</td>
  </tr>
  <tr>
    <td>Priya</td>
    <td colspan="2" style="text-align:center;">Excellent Performance in All Subjects</td>
  </tr>
</table>
```
