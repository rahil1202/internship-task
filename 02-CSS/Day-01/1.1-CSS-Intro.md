
## üé® What is CSS?

**CSS** stands for **Cascading Style Sheets**.
It‚Äôs the language that **styles** your web pages ‚Äî it controls **how HTML elements look** on the screen.

Think of HTML as the skeleton and CSS as the clothes, makeup, and personality you give it.

With CSS, you can:

* Change **colors**, **fonts**, and **sizes**
* Add **spacing**, **borders**, and **shadows**
* Arrange elements on the page (layout)
* Make websites **responsive** (adapt to screen size)

---

## üß± Example: HTML Without and With CSS

### Without CSS:

```html
<h1>Hello, World!</h1>
<p>This is my first website.</p>
```

Result: Plain text. Functional, but boring.

### With CSS:

```html
<h1 style="color: blue; font-family: Arial;">Hello, World!</h1>
<p style="color: gray; font-size: 18px;">This is my first website.</p>
```

Now it has **color**, **style**, and **readability** ‚Äî that‚Äôs the power of CSS!

---

## ‚úçÔ∏è CSS Syntax

CSS follows a simple pattern:

```css
selector {
  property: value;
}
```

Let‚Äôs break it down:

* **Selector** ‚Äî chooses the HTML element you want to style (like `h1` or `.button`)
* **Property** ‚Äî what you want to change (like `color` or `font-size`)
* **Value** ‚Äî how you want it to look (like `blue` or `20px`)

Each rule ends with a semicolon (`;`), and all rules go inside curly braces `{ }`.

### Example:

```css
p {
  color: green;
  font-size: 16px;
}
```

This means:

> All `<p>` elements will have green text and 16px font size.

## üß© Types of CSS (and How to Use Them)

There are **three ways** to apply CSS in a web page:

---

### 1. **Inline CSS**

CSS is written **directly inside the HTML element** using the `style` attribute.

```html
<p style="color: red; font-size: 18px;">This is inline CSS.</p>
```

‚úÖ Quick and simple for small changes
‚ùå Not reusable; messy for big projects

---

### 2. **Internal CSS**

CSS is written inside a `<style>` tag in the `<head>` of your HTML file.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Internal CSS Example</title>
  <style>
    h1 { color: blue; }
    p { color: gray; font-size: 18px; }
  </style>
</head>
<body>
  <h1>Hello CSS</h1>
  <p>This uses internal CSS.</p>
</body>
</html>
```

‚úÖ Keeps HTML cleaner
‚ùå Still not ideal for big websites

---

### 3. **External CSS**

CSS is written in a **separate file** (like `style.css`) and linked using the `<link>` tag.

**HTML file (`index.html`):**

```html
<link rel="stylesheet" href="style.css">
```

**CSS file (`style.css`):**

```css
h1 {
  color: purple;
}
p {
  font-size: 18px;
  color: darkgray;
}
```

‚úÖ Best practice ‚Äî clean, reusable, and easy to maintain
üí° This is how almost every real website uses CSS.

## üß† What Does ‚ÄúCascading‚Äù Mean?

The ‚ÄúCascading‚Äù in CSS means **priority matters** ‚Äî when multiple rules apply to the same element, the browser decides which one wins.

Order of importance:

1. **Inline CSS** (most powerful)
2. **Internal / External CSS**
3. **Browser default styles**

Example:

```html
<p style="color: red;">Hello!</p>
```

Even if your external CSS says `p { color: blue; }`, this inline style will still make the text **red** ‚Äî because it‚Äôs closer to the element.

## üéØ CSS Selectors

Selectors are how CSS **targets elements**.

Here are the main types, with examples:

---

### 1. **Element Selector**

Targets all elements of a specific type.

```css
p {
  color: green;
}
```

‚û°Ô∏è Affects all `<p>` tags.

---

### 2. **ID Selector**

Targets a single, unique element using the `id` attribute.
Prefix it with `#`.

```html
<p id="intro">Welcome to my website!</p>
```

```css
#intro {
  color: blue;
  font-weight: bold;
}
```

‚úÖ Use when you need to style **one specific element**.

---

### 3. **Class Selector**

Targets one or multiple elements that share the same `class`.
Prefix it with a `.` (dot).

```html
<p class="highlight">This is highlighted.</p>
<p class="highlight">This is also highlighted.</p>
```

```css
.highlight {
  background-color: yellow;
  color: black;
}
```

‚úÖ Reusable ‚Äî perfect for repeating styles like buttons or cards.

---

### 4. **Universal Selector**

Applies to *all* elements.

```css
* {
  font-family: Arial;
  margin: 0;
  padding: 0;
}
```

‚úÖ Useful for resetting or applying global styles.

---

### 5. **Grouping Selector**

Style multiple selectors at once.

```css
h1, h2, h3 {
  color: darkblue;
}
```

‚úÖ Saves time and keeps CSS clean.

---

### 6. **Descendant Selector**

Targets elements *inside* other elements.

```css
div p {
  color: red;
}
```

‚û°Ô∏è Only affects `<p>` elements that are **inside a `<div>`**.

---

### 7. **Pseudo-Class Selector**

Used for special states like hover or focus.

```css
a:hover {
  color: orange;
}
```

‚û°Ô∏è Changes link color when the user hovers over it.

---

## üí° Example: Using Different Selectors Together

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>CSS Selectors Example</title>
  <style>
    body { font-family: Arial; }
    h1 { color: darkblue; }
    .highlight { background-color: yellow; }
    #footer { color: gray; font-size: 14px; }
    a:hover { color: red; }
  </style>
</head>
<body>

  <h1>Welcome to My Website</h1>
  <p class="highlight">This paragraph is highlighted using a class.</p>
  <p>Normal paragraph without any class.</p>

  <a href="#">Hover over me</a>

  <div id="footer">
    <p>This is the footer section.</p>
  </div>

</body>
</html>
```

| Concept       | Description                 | Example                         |
| ------------- | --------------------------- | ------------------------------- |
| **CSS**       | Styles HTML elements        | `p { color: red; }`             |
